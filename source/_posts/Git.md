# Git 仓库操作

克隆仓库

```bash
git clone <rep url>
```

设置远程仓库

```bash
git remote add origin git@github.com:iovitz/<repo name>.git
```

# Git 常用指令

暂存

```bash
git add .
```

提交

```bash
git commit -m "<commit message>"
```

回退

```bash
git reset --soft HEAD^
```

cherry-pick

```bash
git cherry-pick <hash>
```

fetch

```bash
git fetch
```
